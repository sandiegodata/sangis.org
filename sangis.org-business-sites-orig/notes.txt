

There are spatialite programs to load in OSM data, which can be used for routing. 

	spatialite_osm_net  -o /Volumes/Storage/Downloads/san-diego-tijuana.osm.pbf  -d sandiego-osm.db -T roads

Creating a routing network in spatialite: 

# Load in the OSM data file
$ spatialite_osm_net  -o /Volumes/Storage/Downloads/san-diego-tijuana.osm.pbf  -d sandiego-osm.db -T roads

# create the roads_net_data table
$ spatialite_network -d sandiego-osm.db  -T roads -t node_to -f node_from -n name -g geometry -c cost -o roads_net_data --overwrite-output

# Create the Virtual Table
$ spatialite sandiego-osm.db 'CREATE VIRTUAL TABLE "roads_net" USING VirtualNetwork("roads_net_data")'

# Try a query
$ spatialite sandiego-osm.db "select * from Roads_Net where NodeFrom = 3079 and NodeTo = 7491;"
